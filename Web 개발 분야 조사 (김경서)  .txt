1. 쇼핑몰

   1) 참고사이트
	- 낫유어로즈 
	  : https://notyourrose.com/
	- 민주킴 공식스토어
	  : https://www.minjukim.co/
	- 유아인바이티드
	  : https://youareinvited.co.kr/

   2) 개선 사항 분석
	- 반응형 비율 조절 문제 ( 작동버튼이 다른 요소에 가려짐 )
	- 버튼 방향 및 폰트 크기 조절 문제 ( 비율적으로 심미적이지 않음 )
	- 흰 배경에 까만 아웃라인으로 과도하게 단순하여 사진요소를 제외하고는 브랜드의 아이덴티디가 보이지 않음
	- SNS 연결버튼의 좌표 조정이 필요함

   3) 개발 방향성
	- 시즌별 룩북 페이지와 상품 판매페이지를 양원화한 구조로 작동해야 함
	- 브랜드의 아이덴티디가 높아질 수 있는 구조가 필요함
	- 관리자 페이지 --> 시즌별 ui 변경이 가능하도록 버튼 선택으로 ui 전체 색감이 바뀌는 기능 추가
	- 사용자 페이지 --> 마이페이지, 장바구니, 제품상세내역 조회가 카테코리별 요소에 최적화 되도록 조정



2. 행사 사이트

   1) 참고사이트
	- 서울일러스트레이션 공식 사이트
	  : https://seoulillustrationfair.co.kr/

   2) 개선 사항 분석
	- 포스터와 텍스트의 비율이 적절하지 않은 화면이 많음 (PC, 모바일 공통)
	- 주요공지 내용의 대부분이 노션으로 연결되어 사이트 내 공지 작성 기능이 부족해보임
	- 플랫한 디자인의 포스터에 비해 버튼이 볼륨 디자인으로 되어 있어 어울리지 않음

   3) 개발 방향성
	- 공식 페이지 내 정보 등록 및 공지가 원활하도록 화면 비율 조정 및 실시간 업데이트 여부가 확인되어야 함
	- 관리자 페이지 --> 공지 등록 및 알림이 원활하도록 기능 추가 및 안정화 필요
	- 사용자 페이지 --> 새로운 공지가 특정기간 동안 다른 색으로 표시되어 행사 참여에 편안함을 느낄 수 있도록 해야 함


3.  매거진 사이트

   1) 참고사이트
	- 디에디트
	  : https://the-edit.co.kr/
	- 라이프집
	  : https://lifezip.kr/magazine

   2) 개선 사항 분석
	- 사진별 해상도 차이로 같은 비율로 적용된 포스트의 퀄리티가 떨어짐
	- PC 및 모바일 환경에서 사진이 효과적으로 노출되도록 비율을 조정해야 함
	- 댓글 기능 및 스크랩 기능을 제공하는 편의성이 필요해보임
	- 일부 포스트 내 사진이 파일명으로 노출되는 오류가 있음

   3) 개발 방향성
	- 태그 및 인덱스로 원하는 카테고리를 손쉽게 접근 가능한 웹매거진의 기능을 활용해야 함
	- 관리자 페이지 --> 조회수에 따라 인기가 높은 글을 상단으로 노출할 수 있도록 조절하는 기능이 추가적으로 필요
	- 사용자 페이지 --> 포스트 작성자를 팔로우, 스크랩하여 자신만의 매거진의 화면이 마이페이지에서 구성되는 기능 추가
