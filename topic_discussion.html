DevnovoS / 강수빈 & 김경서


1. Project Goal

- Java 기반의 FullStack 개발을 위한 WebApp Template 제작한다.
- 용도별 Web의 특징을 자체적으로 분석하여 Problem, Error를 개선하는 방식을 찾는다.
- FrontEnd, BackEnd가 기능적, 심미적으로 어울리는 Web을 제작하는 방식을 연구한다.

 2... GitHub (DevnovoS Organization)
 3... Slack
 4... Figma
 4... Adobe Photoshop & Illustrator

- Language & Database
 1... Java
 2... JavaScript

3. Topics

 1) 도서 콘텐츠 추천 웹메거진
 : Problem - 화면비율 불균형 (배경이미지, 로그인구성)
 : Focusing - 사용자별 맞춤형 튜토리얼창, 카테고리 기능 개선
       업데이트 및 조회수별 콘텐츠 강조 기능 추가

 2) 전시 및 행사 공지 웹페이지
 : Problem - 메뉴별 콘텐츠 배치
 : Focusing - 관리자 페이지 / 운영요소 개선
        사용자 페이지 / 스크랩 및 팔로우 기능 추가

 3) 디자이너 브랜드 쇼핑몰
 : Problem - 기능을 수행할 수 없는 버튼, 가독성이 떨어지는 Header와 Body 구성
 : Focusing - 관리자 페이지 / 시즌별 컨셉을 위한 색상구성변경기능
        사용자 페이지 / 위시리스트 폴더 기능 추가
       Header와 Body 구성 변경, 버튼 기능별 고정화 및 유동화

4. Process & Distribution

 : Slack | 소통 및 개발과정 업데이트
 -> Figma | UI 제작 및 기능 구성 계획
 -> GitHub | 소스코드 관리
 -> AWS | 배포
