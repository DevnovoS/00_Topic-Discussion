DevnovoS / 강수빈 & 김경서


1. Project Goal

- Java 기반의 FullStack 개발을 위한 WebApp Template 제작한다.
- 용도별 Web의 특징을 자체적으로 분석하여 불필요한 요소를 제거하고 인터페이스를 개선한다.
- 기능적, 심미적으로 효과적인 Web을 제작하는 방식을 연구한다.
- 웹콘텐츠 접근성 지침에 따라 다양한 사용자에게 편리한 웹을 제작한다.

3. Topics

 1) 도서 콘텐츠 추천 웹메거진
 : Problem - 화면비율 불균형 (배경이미지, 로그인구성)
 : Focusing - 사용자별 맞춤형 튜토리얼창, 카테고리 기능 개선
       업데이트 및 조회수별 콘텐츠 강조 기능 추가

 2) 전시 및 행사 공지 웹페이지
 : Problem - 메뉴별 콘텐츠 배치
 : Focusing - 관리자 페이지 / 운영요소 개선
        사용자 페이지 / 스크랩 및 팔로우 기능 추가
